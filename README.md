运行方式：

您需要在电脑上有 `go`  语言的编译器

```
go build client.go
go build server.go
```

然后先打开 `server.exe` ，再 打开 `client.exe`  即可进入游戏。

您的 `username:admin, password:12345` 

成功输入账号和密码后，就可以开始游戏了。

游戏说明：

- 按 WASD 进行移动，Q 放弃
